About Me

Halo, saya Dona, seorang mahasiswa yang memiliki minat besar di bidang teknologi, khususnya pengembangan website dan desain UI. Saya terbiasa menggunakan HTML, CSS, dan JavaScript untuk membuat tampilan website yang sederhana namun menarik. Saya senang belajar hal baru dan terus mengembangkan kemampuan saya melalui berbagai latihan dan proyek kecil.

My Journey

Perjalanan saya di dunia teknologi dimulai dari rasa penasaran tentang bagaimana sebuah website dibuat. Dari situ, saya mulai belajar HTML dan CSS di kampus saya, lalu berkembang dengan mempelajari JavaScript untuk menambah fungsi pada website. Setiap proses belajar dan proyek kecil yang saya kerjakan menjadi langkah penting dalam perjalanan saya untuk terus berkembang sebagai web developer.
